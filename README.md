# DICOMViewer

## 概要
UnityでDICOMファイルを閲覧するアプリケーションです。

## 操作方法

|||
|----|----|
|WASDキー|視点の水平移動|
|マウスホイール|ズーム|
|マウスドラッグ|DICOMモデルの回転|
|Rキー|視点のリセット|
|Escキー|終了|
